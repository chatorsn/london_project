1. Создать Next.js 15 с App Router, TypeScript, Tailwind

2. Создать Header с логотипом, навигацией, переключателем EN/RU и кнопкой Book

3. Создать Hero с фоном Лондона и заголовком LONDON ROUTE TRANSFERS

4. Создать Services с 4 услугами: Airport, Corporate, Group, Private

5. Создать About с описанием компании и статистикой (15+ лет, 5 аэропортов, 24/7)

6. Создать CTA с призывом и кнопкой бронирования

8. Создать страницу менеджера с mock-данными, фильтром по статусу и поиском.

9. Добавить анимации Framer Motion (спиннер загрузки, появление строк таблицы с задержкой, hover эффекты).

11. Добавить переводы для всех страниц на русский язык.

Проект соответствует ТЗ этапа 2. Время разработки ~4 дня.
