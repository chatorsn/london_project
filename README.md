# London Route Transfers - Fullstack приложение для бронирования трансферов

## О проекте

Полноценное fullstack-приложение для транспортной компании. Клиенты могут оставлять заявки на трансфер, менеджеры просматривать и управлять бронированиями.

## Технологии

**Фронтенд:**
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- next-intl (EN/RU)
- react-hook-form + zod
- framer-motion

**Бэкенд:**
- Node.js + Express
- SQLite
- CORS

## Запуск проекта

### 1. Запустить бэкенд (порт 4000)

cd backend
npm install
npm run dev

### 2. Запустить фронтенд (порт 3000)

cd ..
npm install
npm run dev

Открыть http://localhost:3000

## Как это работает

Форма бронирования отправляет данные на бэкенд через POST /api/bookings. Бэкенд валидирует данные, сохраняет в SQLite и возвращает ответ. Страница менеджера загружает все бронирования через GET /api/bookings. Можно фильтровать по статусу и искать по имени или ID.

## Структура API

POST /api/bookings - создать бронирование
GET /api/bookings - получить список всех бронирований
GET /api/bookings/:id - получить одно бронирование
PATCH /api/bookings/:id/status - обновить статус

## Особенности

Форма не позволяет выбрать прошедшую дату. Бронирования сохраняются в базу данных и появляются в менеджере. Адаптивная вёрстка под ПК, планшеты и телефоны. Полная локализация на русский и английский языки.
