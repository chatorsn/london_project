# AI Usage Report

## Использованные AI-инструменты

- ChatGPT / Claude

## Что было сделано с помощью AI

1. Генерация первоначальной структуры компонентов (Header, Hero, Services, About, CTA, Footer)
2. Настройка next-intl для двух языков (EN/RU)
3. Создание формы бронирования с react-hook-form и zod
4. Создание страницы менеджера с mock-данными, фильтром и поиском
5. Написание JSON файлов переводов
6. Добавление анимаций Framer Motion

## Что было сделано самостоятельно

1. Сборка и запуск проекта
2. Установка всех зависимостей
3. Настройка Tailwind CSS v4
4. Исправление ошибок сборки и JSON
5. Доработка стилей под нужный дизайн
6. Добавление запрета выбора прошедших дат в форме
7. Настройка плавного скролла
8. Создание страниц privacy, cookies, terms
9. Написание документации

## Сложности, которые возникли

1. Ошибки с next-intl и получением переводов
2. JSON файлы ломались от лишних запятых
3. Порт 3000 был занят, разобралась как очистить
4. Анимации на главной мешали — убрала
5. Tailwind миграция заняла время

## Вывод

AI помог с каркасом проекта. Основная доработка, исправление ошибок и доведение до рабочего состояния сделаны вручную. Проект соответствует ТЗ.
